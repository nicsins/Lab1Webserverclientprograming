Essentials:

Continue working on the favorites website you began in class.
Practice HTML by creating the a web page called favorites.html with the following elements. The sections should be in this order.
  • A h1

    element with the text "Some of my favorite things"

  • A div element for your favorite movie. The div should have the class favorite-section. Inside the div:
    • Add a h2 element with the text "Favorite Movie"
    • Write a short p paragraph description of one of your favorite movies
    • Surround the movie name with a span tag, with an id attribute id="favorite-movie-title". 
    • Add a link to a website about that movie. 
  • Create another div for your favorite band/music artist. The div should have the class favorite-section. Inside this div:
    • Add a h2 elements with the text "My Favorite Band/Artist"
    • Describe your favorite band/artist in a p paragraph. Surround the band/artist name with a span tag, with an id attribute id="favorite-music-artist-name". 
    • Add a link to their website.
  • Create another div for your favorite food. The div should have the class favorite-section. Inside this div:
    • Add a h2 element with text "My Favorite Food".  
    • Add a ul unordered list of your 3 favorite foods
    • Give the li element containing your most favorite food the id = "most-favorite-food"
  • Create another div for some favorite images. The div should have the class favorite-section. Inside this div:
    • Add a h2 with the text "Some Favorite Images"
    • Download two images of your choice from https://unsplash.com/ or https://pixabay.com/ or https://commons.wikimedia.org/wiki/Main_Page, and save in the same directory as your favorites.html file
    • Add two img elements, one for each image
    • Give both img elements the class = "favorite-image"
  • Create another div for your favorite books. The div should have the class favorite-section. Inside this div:
    • Add a h2 element with the text "My Favorite Books"
    • Create a table about your two favorite books.
    • Organize this information into a table with tr, th and td elements to create a header row, and three columns: the book's name, author, and a link to a web page about that book, for example, a page on Amazon. 
Practice CSS by adding styles of your choice to your web page. Create a file called style.css in the same directory as favorites.html, and use a link in favorites.html to include your styles.
  • Make sure all of your section div elements have the same class "favorite-section". Use margins, padding, and borders to arrange and space these sections.
  • Add the class attribute "section-header" to each of your section h2 headers. Write CSS selector and styles to apply a style of your choice to the section headers.
  • Make the background your favorite color. Hint: apply the style to the body element.  If desired, change the color of the body text too. 
  • Use CSS to select your favorite-movie-title span to by the ID.  Use attributes to display the name of the movie in italic text, and a different color to the paragraph text.
  • Use CSS to select your favorite-music-artist-name span by the ID. Display the name of the band in bold text and a different color to the paragraph text.
  • Use CSS to select your most-favorite-food li element. Add a style of your choice to this element. 
  • Use CSS to select your images by the class favorite-image. Add one (or more) of these styles (rounded corners, filters, centering, responsive to page size, apply style on hover) to your images https://www.w3schools.com/css/css3_images.asp
  • Apply styles to your table - display the header <th> elements in one color text, and the body <td> elements in a different color. Add a border.  
Optional: add any other styles you like, add more HTML elements and data.
